# Cognitive Science & LLMs

Course website for the 8001 OMSCS Seminar 2026, built with [Just the Class](https://github.com/kevinlin1/just-the-class).

## Local Development

1. Install Ruby `3.3.6` and Bundler: `gem install bundler`
2. Install dependencies: `bundle install`
3. Run locally: `bundle exec jekyll serve`
4. Open `http://localhost:4000/cog-sci-and-llms/`

## Deployment

Push to `main` — GitHub Pages builds and deploys automatically.
